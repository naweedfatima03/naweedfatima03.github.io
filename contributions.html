Girls Make Games Ambassador

As a Girls Make Games Ambassador, I delivered workshops and hands-on sessions introducing young girls to programming, visual computing, and game development. These workshops focused on creativity, technical confidence, and building early exposure to STEM.

Workshops Conducted

  • Pak Turk High School — Game Design Workshop using Unreal Engine (35+ students)
  • Mindstorm Studios — Game Development Workshop using Unity (20+ students)
  • Learn District · Girls Make Games — Introductory sessions on storytelling and interactive design
